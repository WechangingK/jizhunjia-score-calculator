# 基准价得分计算器 🧮

根据评标规则自动计算基准价和应答人价格得分的桌面工具。

## 功能特性

- **基准价自动计算**：根据应答人数量自动匹配合适的规则（<5家/5-9家/>=10家）
- **两套扣分方案**：支持文字描述方案（高扣0.6/低扣0.3）和公式方案（高扣0.4/低扣0.2），一键切换
- **自动排名**：同分并列排名
- **Excel 导入导出**：支持从 Excel 导入报价数据，计算结果可导出为格式化的 Excel 文件
- **零分保护**：负分自动按 0 分计

## 计算规则

### 基准价 A 计算

| 有效应答人数量 | 规则 |
|--------------|------|
| >= 10 家 | 去掉 2 个最高值和 2 个最低值，剩余求算术平均 |
| 5 ~ 9 家 | 去掉 1 个最高值和 1 个最低值，剩余求算术平均 |
| < 5 家 | 全部报价求算术平均 |

### 得分计算

- B = A 时，得满分 **80 分**
- B > A 时，每高 1% 扣对应系数分
- B < A 时，每低 1% 扣对应系数分
- 所有数值四舍五入保留两位小数
- 负分按 0 分计

### 扣分系数方案

| 方案 | 高于基准价 | 低于基准价 |
|------|-----------|-----------|
| 文字描述（默认） | 每 1% 扣 0.6 分 | 每 1% 扣 0.3 分 |
| 公式 | 每 1% 扣 0.4 分 | 每 1% 扣 0.2 分 |

## 快速开始

### 方式一：下载 exe（推荐）

从 [Releases](../../releases) 页面下载 `JizhunjiaScoreCalculator.exe`，双击即可运行。

### 方式二：源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动程序
python main.py
```

**依赖：** Python 3.12+, PySide6, pandas, openpyxl

## 使用说明

1. 在左侧表格中输入应答人名称和报价，或点击「📥 导入Excel」从文件导入
2. 右上角选择扣分方案
3. 点击「🖩 计算得分」
4. 右侧表格查看结果，点击「📤 导出结果」保存为 Excel

## 开发

```bash
# 打包
pyinstaller build.spec
# 输出: dist/JizhunjiaScoreCalculator.exe
```
